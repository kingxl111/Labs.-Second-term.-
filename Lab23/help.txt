Краткая справка по командам:

1) help - вывод справки по командам
Пример:
help

2) insert ПУТЬ ЗНАЧЕНИЕ. Формат пути: cbcbcbbb, где c - ребенок, b - следующий брат.
Таким образом последовательность cbc... образует путь. 
Значением для данной команды является любой цвет из предложенных в enum colors

Пример: 
insert ccbc yellow

3) print - напечатать дерево в виде файловой системы

Пример:
print

4) delete ПУТЬ. Формат пути: cbcbcbbb, где c - ребенок, b - следующий брат.
Таким образом последовательность cbc... образует путь. 
В результате работы команды в консоль выводится удаленное значение из enum colors

Пример:
delete cbbbcb
Результат:
red

5) task ПУТЬ. Само индивидуальное задание:
Определить число вершин дерева, степень которых совпадает со значением элемента
Путь указывается для элемента, значение которого будет использоваться для сравнения его
со степенями вершин всего дерева.
В результате работы команды в консоль выводится вычисленное количество подходящих вершин дерева.

Пример:
task cbcbcbb 
Результат:
4

Список доступных цветов:
red
green
blue
yellow
purple
orange
white
